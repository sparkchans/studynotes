1. 递归和循环虽然都可以用来解决问题, 但是递归使用的时候需要考虑栈溢出的情况, 防止方法栈的调用过深. 例如在使用递归进行整数的求和时, 当求和数超过 2 ^ 14 次方时就会发生栈的溢出.

2. ```java
    public static int sumByRecursive(int n) {
        if (n == 1) {
            return n;
        }
        return n + sumByRecursive(n - 1);
    }
    
    public static int sumByCycle(int n) {
        int result = 0;
        for (int i = 1; i <= n; i++) {
            result += i;
        }
        return result;
    }
