1. 要求: 输入一个链表的头节点, 从尾到头反过来打印出每个节点的值.

2. 分析: 利用递归打印或者利用栈来保存数据

3. 实现:

   ```java
   public ArrayList<Integer> printListFromTailToHead(ListNode listNode) {
           ArrayList<Integer> result = new ArrayList<Integer>();
           printListFromTailToHead(listNode, result);
           return result;
       }
       public void printListFromTailToHead(ListNode node, ArrayList<Integer> result) {
           if (node == null) {
               return;
           }
           if (node.next == null) {
               result.add(node.val);
           } else {
               printListFromTailToHead(node.next, result);
               result.add(node.val);
           }
       }
   ```
